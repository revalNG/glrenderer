Вот формат файла my.txt, из которого загружаются данные



version 1          - Версия файла
decimalseparator . - Разделить целой и дробной частей

Type Name X Y Z R G B A DirX DirY DirZ UpX UpY UpZ p1 p2 p3 p4

Type - Тип примитива: sphere, cube, cylinder. Пока работают только сфера и цилиндр
Name - Имя (номер) примитива
X, Y, Z - позиция примитива
R, G, B, A - цвет примитива. A - прозрачность, не уверен, что работает
DirX, DirY, DirZ - вектор направления (для сферы не важен)
UpX, UpY, UpZ - вектор "верха" - (для сферы не важен)
p1 p2 p3 p4 - специфические параметры примитива
сфера   - радиус, остальные игнорируются
куб     - длина/ширина/высота, последний инорируется
цилиндр - длина/радиус, остальные игнорируются
